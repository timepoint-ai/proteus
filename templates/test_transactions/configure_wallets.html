Configure Test Wallets

Important: These wallets will be used for test transactions. Only use test networks or wallets with minimal funds.

Required Test Wallet Configuration

To run test transactions, you need to configure the following secrets in Replit:

Secret Name Description Example Format
TEST_WALLET_ADDRESS Main test wallet address for creating markets and submissions 0x1234...abcd
TEST_WALLET_PRIVATE_KEY Private key for the test wallet (keep secure!) 0xabcd...1234
TEST_ORACLE_WALLETS JSON array of oracle wallet addresses ["0xaaaa...","0xbbbb...","0xcccc..."]
TEST_NETWORK_RPC RPC endpoint for test network (optional) https://eth-sepolia.g.alchemy.com/v2/YOUR-API-KEY
TEST_CHAIN_ID Chain ID for test network (optional) 11155111 (Sepolia)
How to Configure:
  1. Click the padlock icon in the Replit sidebar
  2. Add each secret with the exact name shown above
  3. For test purposes, you can use:
    • Sepolia testnet (get free ETH from faucets)
    • Local Hardhat/Ganache network
    • Any EVM-compatible test network
Security Note:
Never use mainnet wallets or wallets containing real funds for testing! These test transactions will consume gas and could result in loss of funds if misconfigured.
